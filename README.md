## Warning!!
The program (including all the required libraries and llama3 7b model pulled with Ollama) is going to take more then 7 GB of a disc space!!

## How to run the code:
- Download, install and run the Ollama. Official website - https://ollama.com/
- Open terminal in a code editor you use and run the comand:
```
ollama pull llama3
```
Wait for Ollama to download the model.
- Open the directory with project, create and activate virtual enviroment
(If you don't know how to do it, here is an instruction: https://docs.python.org/3/library/venv.html).
- Install libraries from **requirements.txt** file by running the comand:
```
pip install -r requirements.txt
```

# You can now run the code by running main.py file
